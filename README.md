**[해결 : Lombok]** <br>
@Entity를 선언하는 Board 에서 @Data를 선언하여 사용했다. <br>
@Data Lombok 을 사용하면 get 메소드를 이용할 수 있으나 메소드 사용 불가 오류 : Lombok 엔티티 전송에 오류가 있다고 판단 <br>
Lombok annotation 재설치 -> Board에서 인식되지 않던 title, content 인식 <br>
